# Create-Node-App Homebrew Tap

Homebrew formulae for [Create-Node-App](https://create-awesome-node-app.vercel.app) CLI tools.

## Install

```bash
brew tap Create-Node-App/tap
brew install create-awesome-node-app
```

## Usage

```bash
create-awesome-node-app my-app
# or
npm create awesome-node-app@latest my-app
```

## Update

```bash
brew upgrade create-awesome-node-app
```
